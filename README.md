# Projet_6
 
Front-end developer of a team working on the design of an interior designer's portfolio site.
Missions: 
- The presentation page of the architect's work (from the HTML provided);
- The site administrator (customer) login page (code to be created from scratch);
- The modal allowing to upload new media (code to be created from scratch).
